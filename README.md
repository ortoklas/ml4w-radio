# ML4W Radio

A simple Internet Radio module for the ML4W Quickshell status bar.

## Features

- Play/Stop internet radio
- Switch between stations
- Lightweight mpv backend
- Configurable station list
- Quickshell integration

## Requirements

- ML4W
- Quickshell
- mpv

## Installation

```bash
./install.sh
```

## Uninstallation

```bash
./uninstall.sh
```

## Configuration

Stations are stored in:

```
~/.config/radio/stations
```
